# E-kata (MVC PHP / MySQL)
E-Kata est un projet de fin d’études visant à digitaliser une marque de vêtements à travers une plateforme e-commerce moderne et intuitive.
Une site e-commerce **MVC** (HTML/CSS/Bootstrap/JS/PHP/MySQL) : produits, recherche + filtres, compte, panier, commande, admin.
